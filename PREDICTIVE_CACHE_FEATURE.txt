HOW IT WORKS
------------

1. FIRST SCAN (Normal Speed)
   - Scans directory normally
   - Caches results automatically
   - Stores in scan_cache.json

2. REPEAT SCAN (20-30% Faster!)
   - Checks cache first
   - Loads cached results instantly
   - Validates cache is still fresh

3. SMART INVALIDATION
   - Detects directory modifications
   - Auto-expires after 7 days
   - Removes outdated entries

WHEN CACHE IS USED
------------------

‚úÖ Cache IS used when:
- Same directory scanned again
- No file extension filter applied
- Directory hasn't been modified
- Cache is less than 7 days old

‚ùå Cache NOT used when:
- First time scanning directory
- File extension filter is active
- Directory has been modified
- Cache has expired
