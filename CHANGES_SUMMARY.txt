RECENT CHANGES SUMMARY
======================

1. NOTIFICATION ENABLE/DISABLE FEATURE
   - Added checkboxes in Settings dialog to enable/disable notifications
   - Email notifications: OFF by default
   - Teams notifications: OFF by default
   - Users must explicitly enable them in Settings
   - Settings saved to .env file automatically

2. CANCEL BUTTON TEXT COLOR FIXED
   - Changed from black to white for better readability
   - Now matches the Export button styling
   - Much clearer on the red background

3. FILE TYPE CATEGORIZATION
   - Added FileExtension column to Excel export
   - Added FileType column with 17 categories:
     * Document, Spreadsheet, Presentation, PDF
     * Image, Design, Video, Audio
     * Archive, Code, Web, Database
     * Executable, Font, 3D/CAD, Email, Other

HOW TO USE NEW FEATURES
========================

Enable Notifications:
1. Click ⚙ Settings button
2. Go to Email or Teams tab
3. Check "Enable Email Notifications" or "Enable Teams Notifications"
4. Fill in your settings
5. Click "Save Settings"

Disable Notifications:
1. Click ⚙ Settings button
2. Uncheck the enable checkbox
3. Click "Save Settings"

IMPORTANT NOTES
===============

- Notifications are OFF by default
- You must enable them in Settings to receive notifications
- Each notification type (Email/Teams) can be enabled independently
- Settings are saved to .env file
- No notifications will be sent unless explicitly enabled

REBUILD REQUIRED
