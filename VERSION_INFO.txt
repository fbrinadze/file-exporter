VERSION INFORMATION
===================

CURRENT VERSION: 2.0

WHERE VERSION IS DISPLAYED
--------------------------

1. WINDOW TITLE BAR
   "File Location Exporter v2.0"

2. LOWER RIGHT CORNER
   "Version 2.0" (gray text, small font)

3. SOURCE CODE
   __version__ = "2.0" (at top of file_exporter.py)


HOW TO UPDATE VERSION
---------------------

When releasing a new version:

1. Edit file_exporter.py
   Find: __version__ = "2.0"
   Change to: __version__ = "2.1" (or whatever version)

2. The version will automatically appear in:
   - Window title bar
   - Lower right corner of GUI

3. Rebuild the executable:
   .\build_exe.bat

VERSION NUMBERING SCHEME
-------------------------

Format: MAJOR.MINOR

MAJOR version (2.x):
- Significant new features
- Breaking changes
- Major redesigns

MINOR version (x.0):
- New features
- Improvements
- Bug fixes
- No breaking changes

Examples:
- 2.0 → 2.1 (minor update, new features)
- 2.1 → 3.0 (major update, significant changes)


DISPLAYING VERSION IN GUI
--------------------------

Location: Lower right corner
Font: Arial, 8pt
Color: Gray
Text: "Version 2.0"

The version label is created in the __init__ method:

    version_label = tk.Label(
        root,
        text=f"Version {__version__}",
        fg="gray",
        font=("Arial", 8)
    )
    version_label.pack(side="right", anchor="se", padx=10, pady=5)
