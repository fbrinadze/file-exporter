FILE LOCATION EXPORTER - FOLDER STRUCTURE
==========================================

MAIN APPLICATION FILES
----------------------
file_exporter.py              - Main GUI application with settings dialog
file_exporter_core.py         - Core scanning and export logic
email_notifier.py             - Email notification module
teams_notifier.py             - Teams notification module

CONFIGURATION
-------------
.env                          - Your settings (NOT in git - created by Settings dialog)
.env.example                  - Template for manual configuration
.gitignore                    - Prevents .env from being committed

BUILD & DISTRIBUTION
--------------------
build_exe.bat                 - One-click executable builder
create_distribution.bat       - Creates distribution package
requirements.txt              - Python dependencies

TESTING
-------
test_notifications.py         - Test email and Teams notifications
test_file_exporter.py         - Unit tests (if present)

DOCUMENTATION
-------------
README.md                     - Quick start guide
DOCUMENTATION.md              - Complete documentation (ALL info in one file)
DISTRIBUTION_README.txt       - Instructions for end users

BUILD OUTPUT (after building)
-----------------------------
dist/
  └── File_Exporter.exe       - Standalone executable

build/                        - Temporary build files (can delete)
*.spec                        - PyInstaller config (can delete)

DISTRIBUTION PACKAGE (after create_distribution.bat)
----------------------------------------------------
File_Exporter_Distribution/
  ├── File_Exporter.exe       - Main program
  ├── DISTRIBUTION_README.txt - User instructions
  ├── .env.example            - Configuration template
  ├── SETUP_EMAIL.txt         - Email setup guide
  └── SETUP_TEAMS.txt         - Teams setup guide

PYTHON CACHE (can ignore)
-------------------------
__pycache__/                  - Python bytecode cache
*.pyc                         - Compiled Python files

GIT (version control)
---------------------
.git/                         - Git repository data
.gitignore                    - Files to exclude from git

WHAT TO SHARE WITH USERS
------------------------
Option 1: Just the .exe
  → dist/File_Exporter.exe

Option 2: Full package (recommended)
  → File_Exporter_Distribution/ (zipped)

WHAT NOT TO SHARE
-----------------
✗ .env (contains your passwords!)
✗ __pycache__/
✗ build/
✗ .git/
✗ Your personal credentials

CLEAN FOLDER STRUCTURE
----------------------
After cleanup, you should have:

Root/
├── file_exporter.py
├── file_exporter_core.py
├── email_notifier.py
├── teams_notifier.py
├── .env (your settings)
├── .env.example
├── .gitignore
├── build_exe.bat
├── create_distribution.bat
├── requirements.txt
├── test_notifications.py
├── README.md
├── DOCUMENTATION.md
├── DISTRIBUTION_README.txt
└── dist/
    └── File_Exporter.exe

TOTAL: ~15 files in root (clean and organized!)
