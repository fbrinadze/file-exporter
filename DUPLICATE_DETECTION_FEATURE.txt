ALGORITHMS USED
---------------

1. Size Grouping
   - Groups files by size
   - Skips unique sizes

3. Full MD5 Hash
   - Complete file hash
   - 100% accurate
   - Only for matches

WASTED SPACE CALCULATION
-------------------------

Formula:
  Wasted = Total Size - (Total Size / Count)

Example:
  3 files √ó 1 MB each = 3 MB total
  Keep 1, delete 2 = 2 MB wasted
